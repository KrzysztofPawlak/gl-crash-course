# gl-crash-course

## Zadanie testowe - zajęcia I
* Stworzyć aplikację zawierająca 2 Activity (Jeden i Dwa) - UI dowolne
* Stworzyć nawigację pomiędzy oboma Activity
* Drugie Activity powinno zawierać 2 Fragmenty
  * Pierwszy fragment widoczny po wystartowaniu Activity,
  * Drugi wyświetlany w miejscu pierwszego po kliknięciu w pierwszy (np. Button
umieszczony w pierwszym fragmencie)
* Zachować informację o ilości przejść pomiędzy Activity Jeden i Dwa
  * Po pierwszym przejściu z Activity Jeden do Dwa powinna być widoczna informacja:
“Ilość przejść : 1”,
  * Po powrocie do Activity Jeden i ponownym przejściu do Dwa powinna być widoczna
informacja: “Ilość przejść : 2”, itd.
  * Informacja o ilości przejść powinna być wyświetlana wewnątrz pierwszego
fragmentu
* Informacja o ilości przejść pomiędzy Activity powinny być zachowane w aplikacji do czasu
jej całkowitego wyłączenia
* Informacja o ilości przejść pomiędzy Activity powinna być zachowana przy zmianie
orientacji aplikacji
* Zadanie bonusowe
  * Dodanie do aplikacji Splash Screena
  
Zachęcamy do eksperymentowania z layoutami, oraz zagadnieniami poruszonymi na pierwszych
zajęciach tj. styles, diemens, strings itd.
